# Wedding RSVP Product Spec

## Goal
Build a wedding RSVP website with guest access, RSVP collection, custom pages, registry, travel info, and admin dashboard.

## Features
- Online RSVP
- Password-protected guest access
- Guest database
- Custom RSVP questionnaire
- Registry page
- About us page
- FAQ page
- Travel page with maps
- Paperless invitations
- Digital save-the-dates
- Email sending
- Seating chart builder
